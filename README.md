# Side Nav

> Side nav with drag and drop functionality built with ReactJS

### Setup Instructions

> 1. Fork this repo
> 1. Clone your forked copy of this repo
>    - `git clone https://github.com/[Your Username]/react_side_nav.git`
> 1. Change directory into the newly cloned repo
>    - `cd react_side_nav`
> 1. Install dependencies 
>    - `npm install`
> 1. Start dev server
>    - `npm start`
> 1. Open a browser and navigate to `localhost:3000` You should see the side nav demo
